# Images folder

Drop your image files here. The website reads them automatically.

- logo.png        -> lab logo (top-left + footer). If missing, the text "AQAR" shows.
- sir.png         -> PI photo (Team section). If missing, an icon shows.
- pub-1.png ..    -> graphical abstracts for the hero Highlights (optional).
                     If missing, a flask icon shows.
- gallery-1.jpg ..-> Gallery photos. Also list them in config.js -> gallery: [...]
                     If gallery is empty, placeholder tiles show.

After adding/changing files here, just refresh the page (no code edits needed).
